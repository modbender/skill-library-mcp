## 7. When-Then Cards

For Transition Mode. Create rules that eliminate the need to decide.

```
🔄 MY WHEN-THENS

When [trigger] → then [action]

Examples to adapt:
- When I finish my coffee → I open my work document
- When the timer goes off → I save and stand up for 2 minutes
- When I leave a meeting → I write 3 notes before doing anything else
- When I feel like I can't go on → permission to stop for 15 minutes
- When I open my phone for no reason → I put it face down and drink water

My when-thens for today:
1. When → then 
2. When → then 
3. When → then 
```

---

## Notes for the Agent

- Templates are starting points. ALWAYS adapt them to the user's context
- Don't present the full template without context — fill it in together
- If the user gets overwhelmed by the template itself → simplify: "Forget the template. What's THE thing today?"
- Emojis are optional — some find them helpful, others find them distracting
- Never insist on completing all fields — "whatever comes out is fine"
