# Compliance Guardrails

Use these checks before sending:

1. Remove guaranteed outcomes:
   - Avoid phrases like "guaranteed growth", "double your income", "passive income guaranteed".
2. Remove fabricated proof:
   - Do not invent downline size, commission gains, or testimonial claims.
3. Respect licensing context:
   - Avoid legal/tax advice.
4. Keep CTA soft:
   - Ask permission for a short call or voice note.
5. Keep message truthful to available profile data:
   - If uncertain, use neutral language.

If a message violates any rule, revise it before send.
