# Output Format Template

## Daily Briefing Format

```
🤖 AI 早报 | {YYYY.MM.DD}（{weekday}）
——————————————

{number}️⃣ {headline_zh}
{summary_zh}（1-2句话）
🔗 {source_url}

...（重复 10-12 条）

——————————————
📌 一句话点评：{daily_insight}
```

## Category Tags
- 🧠 大模型 — Model releases, benchmarks, breakthroughs
- 🤖 Agent — AI agents, automation, tool use
- 💰 融资/商业 — Funding, M&A, partnerships
- 🛡️ 安全/治理 — Safety, regulation, ethics
- 🔧 应用/产品 — Products, features, launches
- 🔓 开源 — Open source models, tools, frameworks

## Formatting Rules
1. Headlines: Concise, informative, Chinese
2. Summaries: 1-2 sentences, key facts only
3. Order: By importance, not chronology
4. Count: 8-12 items per briefing
5. Dedup: No repeated stories across days
6. Links: Include source URL for each item
7. Insight: End with a trend observation connecting 2-3 stories
