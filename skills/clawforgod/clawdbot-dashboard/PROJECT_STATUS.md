## 🚀 How to Use This Project

### Immediate Use
1. Run: `npm install && npm run dev`
2. Open: http://localhost:5173
3. Start using!

### Customization
1. Change colors in `tailwind.config.js`
2. Add messages in `src/data/messages.ts`
3. Modify components in `src/components/`
4. Re-run to see changes (HMR)

### Deployment
1. Build: `npm run build`
2. Deploy: `vercel deploy --prod` or similar
3. Done!

### Integration
1. Read SKILL.md for integration guide
2. Connect Socket.io server
3. Replace dummy data
4. Deploy
