## ⚡ Quick Start

```bash
# Install
cp -r neurocore-ai ~/.openclaw/skills/

# Configure
echo '{"skills": ["neurocore-ai"]}' >> ~/.openclaw/agents/main/agent.json

# Restart
pkill -f "openclaw gateway" && openclaw gateway &

# Test
openclaw skills list | grep neurocore
```
