HOW TO USE
================================================================================

Quick Start:
   1. npm install
   2. cp .env.example .env
   3. Add ELEVEN_LABS_API_KEY to .env
   4. voice-devotional daily --theme peace

CLI Commands:
   • voice-devotional daily [--theme THEME]
   • voice-devotional scripture [--passage PASSAGE]
   • voice-devotional plan [--topic TOPIC] [--days DAYS]
   • voice-devotional roman-road [--voice VOICE]
   • voice-devotional batch [--count NUM]

Programmatic:
   const VoiceDevotion = require('./scripts/voice-devotional');
   const vd = new VoiceDevotion({ apiKey: 'sk_...' });
   await vd.generateDaily({ theme: 'peace' });

Examples:
   • examples/basic.js - Complete usage example
   • examples/batch.js - Batch generation example

Tests:
   npm test
