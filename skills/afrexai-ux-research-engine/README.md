# UX Research Engine ⚡

Complete UX research methodology — from discovery planning to validated design decisions. Zero dependencies. Pure agent skill.

## What This Does

Turns your AI agent into a senior UX researcher. Plan studies, recruit participants, run interviews, build data-driven personas, map user journeys, conduct usability tests, synthesize findings, and validate designs — all with proven frameworks and templates.

## What's Inside

- **Research Planning** — Method selection matrix, sample size calculator, decision tree
- **Recruiting** — Screener templates, quality checklist, channel recommendations
- **User Interviews** — Complete guide template, note-taking YAML, 80/20 moderation rules
- **Persona Building** — Data-driven YAML template with goals/frustrations/behaviors/mental models
- **Journey Mapping** — Stage-by-stage template with emotion curves and service blueprint layer
- **Usability Testing** — Test plan template, task writing rules, severity scale, finding templates
- **Heuristic Evaluation** — Nielsen's 10 scorecard with grading system
- **Research Synthesis** — Affinity mapping process, theme templates, insight formula
- **Research Reports** — Executive summary template, prioritized recommendations
- **Design Validation** — CAMPS critique framework, prototype review checklist
- **Research Ops** — Repository structure, cross-study insight tracker, impact metrics
