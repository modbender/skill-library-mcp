# 男性更年期管理

男性更年期(性腺功能减退症)追踪和管理,包括症状评估、睾酮监测、TRT治疗记录。

## 操作类型

### 1. 记录症状 - `symptom`

记录男性更年期症状。

**参数说明:**
- `info`: 症状描述(必填)
  - 症状类型: libido(性欲)/erectile(勃起)/fatigue(乏力)/mood(情绪)/memory(记忆)/sleep(睡眠)
  - 严重程度: mild(轻微)/moderate(中度)/severe(重度)

**示例:**
```
/andropause symptom libido decreased
/andropause symptom erectile mild
/andropause symptom fatigue moderate
/andropause symptom mood depressed
/andropause symptom memory poor
```

**执行步骤:**

#### 1. 症状分类

**性症状:**
- 性欲减退
- 勃起功能障碍(ED)
- 勃起质量下降
- 晨勃减少或消失

**躯体症状:**
- 体力下降
- 容易疲劳
- 肌肉量减少
- 脂肪增加(尤其腹部)
- 骨密度下降
- 潮热、盗汗

**心理症状:**
- 情绪低落
- 易怒
- 焦虑
- 记忆力下降
- 注意力不集中
- 动力不足

#### 2. 症状严重程度评估

**轻度:**
- 症状轻微,不影响生活
- 偶尔出现

**中度:**
- 症状明显,影响生活质量
- 经常出现

**重度:**
- 症状严重,显著影响生活
- 持续存在

### 2. 记录睾酮水平 - `testosterone`

**参数说明:**
- `info`: 睾酮检测结果(必填)
  - 总睾酮值: 数字
  - 测定时间: HH:mm (可选,如 09:00)
  - 测定日期: YYYY-MM-DD (可选)

**示例:**
```
/andropause testosterone 7.5
/andropause testosterone 7.5 09:00
/andropause testosterone 15.3
/andropause testosterone 总睾酮 7.5 早晨9点
```

**执行步骤:**

#### 1. 解析睾酮信息

**睾酮识别:**
```javascript
patterns = [
  /testosterone[:\s]+(\d+\.?\d*)/i,
  /睾酮[:\s]+(\d+\.?\d*)/i,
  /总睾酮[:\s]+(\d+\.?\d*)/i
]
```

**测定时间识别:**
- "09:00", "早晨9点", "morning"

#### 2. 睾酮水平评估

**总睾酮参考值:**
- 正常: 10-35 nmol/L
- 可能性腺功能减退: 8-10 nmol/L
- 确诊性腺功能减退: < 8 nmol/L(需重复测定)

**测定时机要求:**
- 早晨测定(8-11点)最准确
- 需要至少2次测定确认
- 两次测定间隔>1周

#### 3. 睾酮分类

**性腺功能减退分级:**
```javascript
if (total_testosterone < 8) {
  grade = "confirmed"
  diagnosis = "性腺功能减退症"
  recommendation = "考虑TRT治疗"
} else if (total_testosterone < 10) {
  grade = "possible"
  diagnosis = "可能性腺功能减退"
  recommendation = "结合症状评估,重复测定"
} else if (total_testosterone < 12 && symptoms_present) {
  grade = "borderline"
  diagnosis = "临界性腺功能减退"
  recommendation = "监测症状,定期复查"
} else {
  grade = "normal"
  diagnosis = "睾酮水平正常"
  recommendation = "寻找其他症状原因"
}
```

### 4. TRT治疗记录 - `trt`

记录睾酮替代治疗(TRT)情况。

**参数说明:**
- `info`: TRT治疗信息(必填)
  - action: start(开始)/stop(停止)/effectiveness(效果评估)/side-effects(副作用)
  - 药物信息: type(类型)/dose(剂量)/route(给药方式)

**示例:**
```
/andropause trt start gel 50mg
/andropause trt start 注射 睾酮酯
/andropause trt effectiveness good
/andropause trt side-effects 乳房胀痛
/andropause trt stop
```

**执行步骤:**

#### 1. TRT类型识别

**药物类型:**
- **口服**: 十一酸睾酮
- **注射**: 睾酮酯类(如睾酮庚酸酯、睾酮环戊酸酯)
- **凝胶**: 睾酮凝胶(1%)
- **贴片**: 睾酮贴片

#### 2. TRT治疗评估

**TRT适应症:**
- 总睾酮<8 nmol/L + 症状
- 总睾酮8-12 nmol/L + 明显症状

**TRT禁忌症:**
- 前列腺癌
- 男性乳腺癌
- 未治疗的前列腺增生梗阻
- 红细胞增多症(Hct>54%)
- 严重睡眠呼吸暂停

**相对禁忌症:**
- 前列腺结节
- 心血管疾病高风险
- 肝功能异常

#### 3. 疗效评估

**改善评估时间:**
- 性欲改善: 3-6周
- 勃起功能: 6-12周
