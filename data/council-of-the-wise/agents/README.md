# Council Agents

Drop `.md` files here to add council members. They're auto-discovered.

## Adding an Agent

1. Create `YourAgent.md` with YAML frontmatter (`name`, `emoji`, `domain`) and the agent's persona
2. Include a `## Signature Move` section and `## Example Output` section

That's it. The council will include your new agent on the next run.

## Current Members

- Devil's Advocate (👹) — Challenges assumptions, finds weaknesses
- Architect (🏗️) — High-level strategy, systems thinking, tradeoffs
- Engineer (🛠️) — Implementation, time estimates, concrete steps
- Artist (🎨) — User experience, voice, presentation
- Analyst (📊) — ROI, opportunity cost, quantitative analysis

## Synthesis

- Synthesis (⚖️) — Moderator voice, not a summary. Gives a verdict.
