## Writing Style Quick Reference

**Hook formulas:**
- "Most people get [X] wrong..."
- "I spent [time] learning [X]. Here's what I wish I knew..."
- "Stop doing [common mistake]."
- "[Contrarian opinion]."

**Structure:**
```
Hook (stop the scroll)
↓
Context (why this matters)
↓
Value (the insight/lesson)
↓
Proof (example/story)
↓
CTA (what to do next)
```
