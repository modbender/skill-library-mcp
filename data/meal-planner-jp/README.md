# Meal Planner JP

🍱 Japanese meal planning with nutrition balance and budget optimization.

## Features

- Weekly/daily meal plans with Japanese cuisine
- Budget-conscious (default ¥500/meal)
- Nutrition balanced (主食・主菜・副菜・汁物)
- Shopping list with estimated prices
- Seasonal ingredients (旬の食材)
- Cooking time estimates
- Beginner-friendly recipes

## Usage

- "今週の献立考えて"
- "500円以内で晩ごはん"
- "一人暮らし 1週間の献立"
