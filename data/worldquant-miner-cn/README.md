# WorldQuant Miner CN

WorldQuant Alpha 挖掘器 - AI 驱动的 Alpha 因子生成系统。

## 功能

- ⛏️ Ollama 本地 LLM（3-5x 更快）
- 🧪 自动测试和提交
- 🖥️ Web Dashboard
- 🐳 Docker 部署

## 使用

```bash
# 配置凭据
echo '["email", "password"]' > credential.txt

# 启动
docker-compose -f docker-compose.gpu.yml up -d
```
